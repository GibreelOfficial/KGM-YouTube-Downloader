---

## 🧩 Features

- 🎥 Download videos 
- 📦 Full **playlist support** with individual progress
- ✅ Checkmark shows completed downloads
- 🔍 Fetch video info & formats before downloading
- ⚙️ Built-in **progress bar**
- 📁 Automatically saves to your chosen folder
